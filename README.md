## Estructura del repositorio

La idea de este repositorio es organizar todo el trabajo del semestre en las
siguientes secciones principales:

- `Parciales/`
- `Laboratorios/`
- `Quizzes/`
- `Proyecto_Final/`

Estructura sugerida:

```text
.
├── Parciales/
├── Laboratorios/
├── Quizzes/
├── Proyecto_Final/
└── README.md
```

## Objetivo

Mantener en un solo lugar el material del curso, incluyendo entregas, notebooks,
informes y avances del proyecto final, con una organización clara y consistente.
